# 🛒 Walmart Sales Data Analysis

## 📌 Project Overview
This project analyzes Walmart sales data to understand sales trends, customer purchasing behavior, and overall business performance. The analysis helps identify top-performing branches, popular product lines, and factors that influence sales.

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-selling product lines
- Compare sales across different branches
- Understand customer purchasing patterns
- Visualize sales trends using charts

## 📊 Dataset
The dataset includes the following features:
- Invoice ID
- Branch
- City
- Customer Type
- Gender
- Product Line
- Unit Price
- Quantity
- Total Sales
- Date
- Payment Method
- Gross Income
- Rating

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Jupyter Notebook

## 🔍 Analysis Steps
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization
5. Insight Generation

## 📈 Key Insights
- Identified the branch with the highest sales.
- Found the most popular product categories.
- Analyzed customer ratings and satisfaction.
- Compared different payment methods used by customers.

## 📌 Conclusion
The analysis provides insights into sales patterns and customer behavior, helping businesses make better data-driven decisions.
